# Ziddzide Blog 🔒

> 网络安全 · 渗透测试 · 网络工程 · 运维实践

基于 [Mizuki](https://github.com/LyraVoid/Mizuki) 主题构建的 Astro 静态博客。

## 🚀 本地开发

```bash
pnpm install
pnpm dev      # http://localhost:4321
pnpm build    # 构建
pnpm preview  # 预览
```

## 📝 文章格式

```yaml
---
title: 文章标题
published: 2026-07-15
description: 文章简介
tags: [网络安全, 渗透测试]
category: 技术
draft: false
pinned: false
---
```

## 🛠 技术栈

- **框架**: Astro v7 + Svelte 5
- **样式**: Tailwind CSS v4
- **搜索**: Pagefind
- **代码高亮**: Expressive Code
- **部署**: GitHub Pages (Actions)
